# solidity-solids
A collection of solid smart contracts and libraries for Solidity


#### sample:
* RoleBased (and Has<*> specializations)
* Switchable
* Loggable
* Validating
* Math
* Storage (and its components GenericStorage & GenericArrayStorage)
* financial: Bank & Accountant


### Prerequisites:

```
truffle 4.1.7
node 8.9.x
```

### build
```
    npm install
    npm test
```
