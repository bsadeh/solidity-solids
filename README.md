# solidity-solids
A collection of solid smart contracts and libraries for Solidity


#### sample:
* RoleBased (and Has<*> specializations)
* Switchable
* Loggable
* Validating
* Math
* GenericStorage
* financial: Bank & Accountant


### Prerequisites:

```
truffle 4.0.6
node 8.9.4
```

### build
```
    npm install
    npm test
```
